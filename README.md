# Chora Release

Official distribution channel for Chora CLI and Desktop releases.

## Install Chora CLI

**macOS / Linux:**
```bash
curl -fsSL https://raw.githubusercontent.com/Chora-Inc/Chora-Release/main/scripts/install.sh | bash
```

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/Chora-Inc/Chora-Release/main/scripts/install.ps1 | iex
```

## Releases

CLI and Desktop binaries are published as [GitHub Releases](https://github.com/Chora-Inc/Chora-Release/releases).
